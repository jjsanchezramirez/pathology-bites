# Product Overview

Pathology Bites is a modern, AI-powered educational platform providing free, high-quality pathology learning materials for medical students, residents, and practicing pathologists.

## Core Purpose
- Comprehensive pathology question bank across all subspecialties
- Interactive quiz system with multiple learning modes
- Case-based learning with high-quality pathology images
- Performance analytics and progress tracking

## Key Features
- **Question Management**: Expert-curated questions with detailed explanations, version control, and review workflow
- **Quiz System**: Multiple modes (tutor, timed, untimed) with immediate feedback and adaptive learning
- **Image Management**: High-resolution pathology images with advanced viewer (zoom, pan, annotations)
- **User Roles**: 4-tier system (Admin, Creator, Reviewer, User) with role-based access control
- **Analytics**: Performance tracking, learning paths, and progress monitoring

## Target Users
- Medical students studying pathology
- Pathology residents in training
- Practicing pathologists for continuing education
- Medical educators and content creators

## Educational Focus
- Anatomic and clinical pathology subspecialties
- Cell identification and morphology
- Case-based learning with real pathology images
- Evidence-based explanations with literature references
